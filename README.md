# ThenTheme
manager application theme style， such as color、font、image...



## Installation

#### Cocoapods

```
pod 'ThenTheme'
```



#### Carthage

```
github "ghostcrying/ThenTheme"
```

###### 校验Carthage

```
carthage build --no-skip-current --platform ios --use-xcframeworks
```
